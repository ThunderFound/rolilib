## Installation

```bash
pip install rolilib
```
    
## Examples

### Items name and RAP
```python
import rolilib

item = rolilib.item('138932314')

print('Name:', item.name)
print('RAP:', item.rap)
```
Output:
```bash
Name: Dominus Aureus
RAP: 4316150
```
